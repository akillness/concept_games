# Removed Reference Assets

이 폴더는 이번 아트 정리 과정에서 Unity 프로젝트 `concept_game/Assets` 밖으로 이동한 샘플, 데모, 원본, 참고용 리소스를 보관한다.

## Purpose

- Unity 프로젝트에서는 제외
- 필요 시 복구 가능하도록 보관
- 최종 확인 후 완전 삭제 대상 식별

## Current Contents

- `root/`: 프로젝트 톤과 맞지 않는 전체 팩 또는 고아 메타
- `ui/`: UI 팩 샘플 씬
- `characters/`: PSD 원본, 예제 씬, 캐릭터 데모 파츠
- `environment/`: 환경 팩 샘플 씬, 원본 소스, URP 전용 지원 파일, 홍보 파일
