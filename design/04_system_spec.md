## 1. 플레이어 이동/행동

### 입력

- 이동: WASD / 좌스틱
- 주 행동: 진공 정화기
- 보조 행동: 물살 분무기 또는 장착 모듈
- 궁극 행동: 환경 과부하 정화
- 상호작용: 오브젝트 활성화 / 수거 / 대화

### 상태

- Idle
- Move
- Clean
- Spray
- Interact
- Hurt
- Overdrive

### 구현 규칙

- 플레이어는 항상 탑다운 시점 기준으로 이동
- 이동 중 주 행동 가능
- 보조 행동은 짧은 쿨다운
- 피격 시 넉백은 약하게, 제어 상실 시간은 0.25초 이하

## 2. 정화 시스템

### 오염 단위

오염은 `Corruption Tile`과 `Corruption Node` 두 계층으로 관리한다.

- Tile: 바닥 오염. 정화 진행도만 가짐.
- Node: 핵심 오염체. 주변 타일을 재오염시킴.

### 정화 규칙

- 타일은 정화 게이지가 100%가 되면 회복 상태로 전환
- 노드는 HP가 0이 되면 비활성화
- 노드가 살아 있으면 주변 타일 재오염 확률 발생

### 공식

- 타일 정화 속도 = 기본 정화력 x 툴 배율 x 희귀 버프
- 노드 재오염 주기 = 기본 6초 - 지구 난이도 보정

## 3. 오염 확산 시스템

### 목적

- 플레이어가 "무작정 다 닦는" 대신 우선순위를 판단하게 만든다.

### 규칙

- 모든 타일이 확산하는 것이 아니라 특정 노드만 확산
- 한 섹터의 활성 노드는 최대 3개
- 재오염은 플레이어가 막 닦은 곳을 즉시 뒤엎지 않도록 지연 적용

## 4. 자원 회수 시스템

### 자원 타입

- Bloom Dust: 기본 통화
- Scrap: 구조물 복원 재료
- Seed Pod: 정원/식생 복원 재료
- Clean Water: 장비 충전 및 일부 복원 재료
- Memory Pearl: 희귀 메타 재료

### 드랍 규칙

- 타일 정화: Bloom Dust 확률 드랍
- 노드 제거: Scrap, Seed Pod, Clean Water 드랍
- 숨겨진 오브젝트: Memory Pearl 또는 장식 설계도

## 5. 원정 목표 시스템

각 섹터는 아래 데이터를 가진다.

- 메인 목표 타입
- 목표 수량
- 보조 목표 리스트
- 시간 압박 여부
- 환경 위험 리스트

메인 목표 완료 시 탈출 게이트 활성화.

## 6. 허브 복원 시스템

### 허브 구역

- Dock
- Market
- Greenhouse
- Housing
- Vista Garden

### 복원 단계

- Locked
- Cleared
- Functional
- Upgraded
- Flourishing

복원 단계가 올라갈수록:
- NPC 입주
- 기능 해금
- 장식 슬롯 증가
- 일일 주문 수 증가

## 7. 업그레이드 시스템

### 업그레이드 트리

- Tool Power
- Capacity
- Cooldown
- Movement
- Rare Drop Chance
- Hub Utility

### 규칙

- 전투 계수가 아니라 "정화 효율" 위주
- 한 번에 3단계 이상 급격히 강해지지 않도록 구간 제한
- 각 지구 종료 시 새 업그레이드 브랜치 1개 해금

## 8. 주민/퀘스트 시스템

### 주민 종류

- 선장
- 정원사
- 기계공
- 상인
- 조명 장인

### 퀘스트 종류

- 자원 납품
- 특정 지구 복원
- 특정 생물 구조
- 장식 배치
- 고유 이벤트 완료

## 9. 장식/배치 시스템

장식은 완전 자유 배치가 아니라 `소형 슬롯 / 중형 슬롯 / 특수 슬롯` 기반으로 제한한다.

이유:
- 구현 비용 절감
- 카메라 가독성 유지
- 충돌/내비 문제 감소

## 10. 저장/진행 시스템

저장 항목:
- 해금 지구
- 지구별 별점/클리어 상태
- 허브 복원 단계
- 업그레이드 레벨
- 주민 상태
- 장식 배치 상태
- 인벤토리 자원
- 옵션 설정
