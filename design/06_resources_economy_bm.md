# 06. Resources, Economy, and BM

## 자원 경제 목표

- 복잡한 환율표 대신 5개 핵심 자원으로 관리
- 플레이어가 "무엇이 부족한지" 즉시 이해할 수 있어야 함
- 후반에도 모든 자원이 죽지 않도록 사용처를 유지

## 자원 정의

| 자원 | 용도 | 획득처 | 희소성 |
|------|------|--------|--------|
| Bloom Dust | 범용 통화 | 대부분의 정화 행동 | 낮음 |
| Scrap | 구조물/기계 복구 | 노드 제거, 잔해 회수 | 중간 |
| Seed Pod | 정원/생태 복원 | 식생 정화, 보조 목표 | 중간 |
| Clean Water | 장비/정원 보조 | 펌프, 물 정화, 이벤트 | 중간 |
| Memory Pearl | 희귀 해금 | 결전 섹터, 주민 퀘스트 | 높음 |

## 소비처

### Bloom Dust

- 기본 업그레이드
- 일반 장식 구매
- 허브 상점 주문 처리

### Scrap

- 허브 시설 복구
- 기계 관련 업그레이드

### Seed Pod

- 식생 복원
- 생물 유인 장식

### Clean Water

- 일부 고급 장비
- 특수 정화 이벤트

### Memory Pearl

- 지구 해금 핵심 조건
- 고급 모듈
- 엔딩 직전 최종 복원

## 경제 곡선

### 초반

- Bloom Dust 중심
- Scrap이 첫 병목

### 중반

- Seed Pod와 Clean Water 균형 필요
- 허브 복원과 업그레이드 사이 선택 유도

### 후반

- Memory Pearl과 특수 조건이 병목
- 기본 자원은 플레이어가 이미 충분히 다루고 있어야 함

## 가격 설계 원칙

- 신규 기능 해금은 항상 2~3회 원정 내 달성 가능해야 함
- 지구 진행을 막는 벽은 단일 자원이 아니라 `행동 숙련 + 자원 조합`이어야 함
- 장식 비용은 기능성 투자보다 약간 낮게 배치해 허브 꾸미기를 포기하지 않게 함

## BM 제안

### 본편

- Steam 기준 권장가: USD 12.99
- 할인 전략: 10~20% 초기 런치 할인 가능

### 사후 확장

- OST
- `Harbor Decor Pack` 코스메틱 DLC
- 무료 기념 맵 1개

### 배제 정책

- 하드 커런시 없음
- 스태미나 없음
- 로그인 보상 없음
- 강제 광고 없음

## 왜 프리미엄인가

- 본작은 8시간 완결형 구조라 세션 리텐션보다 경험 밀도가 중요하다.
- F2P 구조를 넣는 순간 자원 곡선이 플레이 만족보다 장기 체류 최적화에 종속된다.
- 솔로/소규모 제작에서는 유료 본편이 더 일관된 품질 목표를 만든다.
