# 07. Rules, Balance, and Fail States

## 기본 규칙

1. 원정은 메인 목표 완료 후 언제든 탈출 가능
2. 플레이어 HP가 0이 되면 강제 귀환
3. 강제 귀환 시 획득 자원의 70%를 보존
4. 메모리 자원은 원정 실패 시 절반만 보존
5. 지구 클리어 별점은 `시간`, `정화율`, `서브목표`, `피격 횟수`로 산정

## 밸런스 목표

- "편안하지만 무의미하지는 않다"
- 너무 어렵게 죽이는 대신, 정화 효율이 떨어지게 만든다
- 액션 숙련보다 루트 판단과 도구 선택이 중요하다

## 난이도 축

| 축 | 쉬움 상태 | 어려움 상태 |
|----|-----------|-------------|
| 오염 밀도 | 노드 2개 | 노드 3개 + 재오염 증가 |
| 지형 장애 | 거의 없음 | 이동 우회 필요 |
| 환경 위험 | 약함 | 동시 처리 필요 |
| 요구 자원 | 단일 자원 중심 | 자원 조합 요구 |

## 적/위험 설계 원칙

- 적은 다수의 체력 덩어리로 만들지 않는다.
- 정화 흐름을 방해하는 유닛 위주로 설계한다.
- 플레이어를 즉사시키는 패턴은 넣지 않는다.
- 카메라 시점에서 읽기 어려운 투사체는 사용하지 않는다.

## 실패 상태

### 원정 실패

발생 조건:
- HP 0
- 핵심 오브젝트 파괴
- 특정 미션의 타이머 실패

결과:
- 허브 귀환
- 일부 자원 손실
- 진행도는 부분 반영

### 시스템 실패 금지 항목

- 장시간 파밍 후 완전한 무보상
- 저장되지 않는 허브 변경
- 지구를 다시 처음부터 해야만 하는 구조

## 별점 규칙

- 1성: 메인 목표 완료
- 2성: 메인 목표 + 서브 목표 1개 또는 정화율 80% 이상
- 3성: 메인 목표 + 서브 목표 달성 + 피격/시간 조건 충족

별점 보상:
- 장식 설계도
- 희귀 씨앗
- 추가 대사 이벤트

## 수치 초기안

- 플레이어 기본 HP: 100
- 기본 이동속도: 4.5
- 기본 정화력: 10/초
- 보조 스킬 쿨다운: 8초
- 궁극기 충전 시간: 평균 90초

## 튜닝 우선순위

1. 원정 길이
2. 정화 만족감
3. 귀환 후 업그레이드 빈도
4. 후반 자원 병목
5. 3성 달성 난이도
